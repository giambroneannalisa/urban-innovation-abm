# Urban Innovation ABM

## Academic Affiliation
PhD: Annalisa Giambrone | Supervisor: Prof. Raffaele Scuderi
Institution: University of Enna Kore | Cycle XXXVIII (2026)

## Overview
Multi-objective optimization of urban innovation using ABM and NSGA-II.

## Key Results
Innovation-Diversity: r=-0.424 (strong trade-off)
Innovation-Equity: r=+0.347 (weak relationship)
86% Rule: 58 solutions achieve high innovation + low inequality

## Quick Start
Prerequisites: NetLogo 6.x + Python 3.8+


**Installation:**
```bash
git clone https://github.com/giambroneannalisa/urban-innovation-abm.git
cd urban-innovation-abm
pip install -r requirements.txt
python nsga2_optimization.py nsga2_config_final.json
```

## Structure
- Urban_Innovation_Model_vFinal_english.nlogo (ABM)
- nsga2_optimization.py (Optimizer)
